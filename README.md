# [작품 소개]

    코로나19로 인해 바깥 외출이 부담스러운 요즘, 커피 한잔을 위해 카페에 방문하는 것을 지양하기위해 집에서 커피를 직접 내려먹는 '홈카페'가 유행이다.
    홈카페를 하기위해 필요한 장비, 커피의 핵심인 나라별 원두, 커피 내리는 방법등의 정보를 담고있는 사이트이다.
